A Lens protocol (https://lens.dev/) browser.
Created as a part of LFGrow Hackathon March 2022

## Getting Started

Install dependencies

```bash
npm install
```

Then run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
